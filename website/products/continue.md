# Continue

Continue is an open-source AI code assistant that allows you to use your own AI models or choose from various providers. It offers flexibility and customization for developers who want control over their AI coding tools.

## Key Features

- **Open Source**: Fully open-source and customizable
- **Model Flexibility**: Use any LLM (OpenAI, Anthropic, local models)
- **Chat Interface**: Ask questions about your code
- **Refactoring**: AI-assisted code improvements

## Pricing

Free and open-source. You only pay for the AI model API usage if using cloud providers.
