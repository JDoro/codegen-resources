# OpenAI Codex

OpenAI Codex is the AI system that powers GitHub Copilot. It translates natural language to code and is available via API for custom integrations.

## Key Features

- **Natural Language to Code**: Describe what you want in plain English
- **API Access**: Integrate into your own tools and workflows
- **Multi-language**: Supports many programming languages
- **Flexible**: Use for various coding tasks

## Pricing

Available through OpenAI's API with usage-based pricing.
