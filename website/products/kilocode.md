# KiloCode

KiloCode is an AI-powered code generation platform that provides access to over 500 AI models from 60+ providers with transparent, usage-based pricing. It offers flexible deployment options including local execution and cloud-based models.

## Key Features

- **Multiple AI Models**: Access to 500+ models from providers like OpenAI, Anthropic, Google, and more
- **Autocomplete & Code Suggestions**: Context-aware inline suggestions as you type
- **Task Automation**: Automate repetitive coding tasks including setup, testing, and browser automation
- **Parallel Agents**: Multiple AI agents can work on complex tasks simultaneously
- **Code Review & Refactoring**: Automatically refactor code, generate documentation, and fix bugs
- **Privacy-Focused**: No training on your code, bring your own API keys, and local execution support
- **MCP Server Marketplace**: Extensible through marketplace and local LLMs support (Ollama, LM Studio)

## Pricing

KiloCode offers transparent, pay-as-you-go pricing with three tiers:

- **Individual**: $0/month with no subscription. Pay only for usage. Includes $15-$25 in free tokens for new users
- **Teams**: $15/user/month plus usage costs. Includes analytics, team management, and priority support
- **Enterprise**: Custom pricing with private gateways, SSO, audit logs, and dedicated support

No markup on AI model costs - charges match provider list prices.
