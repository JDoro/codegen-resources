# Kiro

Kiro is an AWS-developed agentic IDE that prioritizes specification-driven development, transforming prompts into detailed specifications, requirements, design documents, and actionable tasks. Built on VS Code OSS, it bridges the gap between prototype and production code.

## Key Features

- **Spec-Driven Development**: Transforms prompts into detailed specifications, requirements, and design documents
- **Agentic Assistance**: AI agents autonomously generate code, update documentation, create project plans, and run tests
- **Documentation Automation**: Automatically generates and updates technical documents, project plans, and task lists
- **Hooks and Automation**: Responds to file saves and code changes with automated triggers for testing, security checks, or CI
- **Steering Rules**: Guide Kiro's behavior and enforce coding standards via markdown steering files
- **VS Code Compatibility**: Based on VS Code OSS, supports VS Code extensions and easy migration
- **Enterprise Features**: SAML/SCIM SSO, centralized billing, analytics, and advanced security

## Pricing

Kiro offers four main pricing tiers:

- **Kiro Free**: $0/month - 50 credits with basic features
- **Kiro Pro**: $20/month - 1,000 credits plus pay-as-you-go ($0.04 per credit)
- **Kiro Pro+**: $40/month - 2,000 credits plus pay-as-you-go ($0.04 per credit)
- **Kiro Power**: $200/month - 10,000 credits with enterprise features (SSO, analytics, advanced security)

New users receive a bonus of 500 credits valid for 30 days. Enterprise plans with custom pricing are also available.
