# Replit AI

Replit AI is an AI-powered coding assistant built into the Replit online IDE. It helps you code faster with completions, explanations, and debugging assistance.

## Key Features

- **Integrated Experience**: Built directly into Replit
- **Code Completion**: Smart suggestions as you type
- **AI Chat**: Ask questions about your code
- **Debugging Help**: AI-assisted bug finding and fixing

## Pricing

Available with Replit's subscription plans, offering free and paid tiers.
