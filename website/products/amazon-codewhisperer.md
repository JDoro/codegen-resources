# Amazon CodeWhisperer

Amazon CodeWhisperer is an AI coding companion that helps you build applications faster by providing code suggestions based on your comments and existing code.

## Key Features

- **Real-time Suggestions**: Get code recommendations as you type
- **Security Scanning**: Built-in security vulnerability detection
- **Reference Tracking**: See sources for code suggestions
- **AWS Integration**: Optimized for AWS services

## Pricing

Free for individual use with a Professional tier available for advanced features and team collaboration.
