# Lovable

Lovable is an AI-powered full-stack app builder that turns natural language prompts into production-ready code and functional web applications. It operates as a chat-based, in-browser IDE with integrated editing, code export, and real-time collaboration features.

## Key Features

- **Prompt-Driven Development**: Build complete web apps with authentication, dashboards, and backend logic from a single prompt
- **Visual Editor**: Drag-and-drop interface for UI design with real-time previews
- **GitHub Integration**: Export, version, and modify generated code directly in your repositories
- **Reusable Modules**: Quick integration for authorization, analytics, payments, and more
- **One-Click Deployment**: Host apps via Lovable's infrastructure or export for custom deployment
- **Team Collaboration**: Real-time collaboration with permissions management, version control, and commenting
- **Enterprise Security**: SOC 2 compliance, SSO, role-based access, audit logs, and data backup

## Pricing

Lovable offers four pricing tiers:

- **Free**: 5 daily credits (up to 30/month) with public projects and unlimited collaborators
- **Pro**: $25/month - 100 monthly credits with private projects, custom domains, and premium templates
- **Business**: $50/month - Includes SSO, advanced permissions, and data training opt-out
- **Enterprise**: Custom pricing with on-premise deployment and dedicated support

Student discount available (up to 50% off Pro plan).
