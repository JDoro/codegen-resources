# Windsurf

Windsurf is Codeium's AI-powered IDE that introduces agentic flows, allowing AI to autonomously complete complex coding tasks across multiple files.

## Key Features

- **Agentic AI**: AI that can complete complex multi-step tasks
- **IDE Integration**: Full-featured standalone IDE
- **Multi-file Operations**: Work across entire codebases
- **Smart Context**: Deep understanding of your project
