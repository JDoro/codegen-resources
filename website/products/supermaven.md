# Supermaven

Supermaven is an AI code completion tool that stands out with its massive 1 million token context window, allowing it to understand and work with large codebases effectively.

## Key Features

- **Massive Context**: 1 million token context window
- **Ultra-fast**: Optimized for speed and responsiveness
- **Smart Completions**: Context-aware suggestions
- **Large Codebase Support**: Works well with big projects

Supermaven offers a free tier and a Pro plan with enhanced features.
