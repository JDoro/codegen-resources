# Cursor

Cursor is an AI-first code editor designed to make you extraordinarily productive. It's built as a fork of VS Code with native AI integration.

## Key Features

- **AI-First Design**: Built from the ground up with AI at its core
- **Codebase Understanding**: AI that understands your entire codebase
- **Multi-file Editing**: Make changes across multiple files simultaneously
- **Native Chat**: Built-in chat interface for code discussions

## Pricing

Cursor offers a free tier with limited features and a Pro plan for advanced capabilities.
